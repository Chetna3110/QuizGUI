# 🧠 Quiz GUI

An interactive **Quiz Application** built with Java that tests your knowledge through a clean graphical interface.

---

## 📌 About

QuizGUI is a Java-based quiz application with a graphical user interface. It presents multiple-choice questions to the user, tracks the score, and shows the final result at the end. Great for learning and testing knowledge on any topic!

---

## ✨ Features

- Multiple choice questions (MCQs)
- Instant feedback on correct/wrong answers
- Score tracking throughout the quiz
- Final score displayed at the end
- Next question navigation
- Restart quiz option
- Clean and simple GUI

##  How to Run

1. Clone the repository
```bash
git clone https://github.com/Chetna3110/QuizGUI.git
cd QuizGUI
```

2. Compile the Java file
```bash
javac QuizGUI.java
```

3. Run the application
```bash
java QuizGUI
```

> Make sure you have **Java JDK** installed. Check with `java -version`

---

## How to Play

1. Launch the application
2. Read the question displayed on screen
3. Click on your answer (A, B, C, or D)
4. See if you got it right ✅ or wrong ❌
5. Click **Next** to go to the next question
6. At the end, your **final score** is displayed
7. Click **Restart** to play again!

---

## 📊 Scoring

| Result | Points |

| Wrong Answer | 0 points |
